Stylesheet Information

This page relies exclusively on HTML stylesheets for formatting. If you don't have a browser capable of using stylesheets, or you have stylesheet capabilities disabled, you will see nothing but default text. If this is the case, you should enable stylesheet capabilities, or download the latest version of Microsoft Internet Explorer or Netscape Navigator.

Since this page was built to look perfect on Microsoft Internet Explorer 5.01, this is the browser I recommend using. However, this page has been tested on Netscape Navigator 4.7, and this page looks similar.

I understand that you may want to use a custom stylesheet to view the page in a different format. (This is what I would be inclined to do.) To make things easy on you, I will tell you that the pages use the "class" declaration in <P> tags. For example, the tag on this paragraph is <P class="normal">.

The stylesheet uses the following class declarations. Each list item is styled according to its definition (to make things easier to understand).

    .title

    .normal

    .biglink

    .medlink

    .link

    .largetext

    .medtext

    .tinytext

If you prefer, you can just look at the stylesheet to see what I've done.

More help on stylesheet coding is available from Webmonkey's Stylesheets Guide.
