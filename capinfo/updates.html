Civil Air Patrol Information Updates

25 January 2000

  • After some trouble with a DHCP-enabled server, a server with a static IP address was found. This will be the permanent home of Civil Air Patrol Information. Please excuse us as we update the Yahoo! links.

2 January 2000

  • Music is back! However, the music is not the familiar TOOL, it is a combination of old showtunes and classic rock (a slight twist on a failed theme, but the webmaster prefers this). Notice a cohesion in the music: what is the common theme of the songs on the three "Missions" pages?

  • In order to make room for the music, Javascript windows have been disabled.

28 November 1999

  • The page has undergone a complete renovation, eliminating all Java applets, and becoming Javascript-enabled. Music is removed (temporarily), all links are fixed (finally). HTML stylesheets are added, allowing users to customize the appearance of Civil Air Patrol Information

Spring, Summer 1999

  • The site has been moved to GeoCities, and then shifted around some more. Renovations were made, then undone, and then made again. The webmaster has not kept the updates page well-updated.

24 February 1999

  • The "Credits" Window has been revised and given the new name NeoCredits. The applet features better textual display, superior arrangment, copy capability (using CTRL-C or another key combination), and a user-customizable color scheme. Flickering has been fixed.

19 February 1999

  • CAP Info releases the "Credits" applet, which displays basic credit information in a separate window. The applet features a pull-down menu bar. A button on the main hypertext page will revive the window if it is closed.

8 February 1999

  • CAP Info becomes "Java IN-Compatible"! The addition of a Java applet on the main page outlines the evolution of Java applets being designed for CAP Info. Keep checking back for improved, more relevant applets!

5 February 1999

  • Background images revised to improve text readability. TOOL sounds are back! After receiving complaints about the mix of classical music and show tunes, CAP Info has decided to return to the original TOOL music, with a few additions / alterations.

4 February 1999

  • Background graphics and main page images completely renevated. Image map removed, navigation frame moved to the left.

  • CAP Info Updates page added, allowing repeat viewers to see changes in one location.
